## Indexer Contract

Contract indexer collects the info about NEAR contract state changes

### Config

```
DATABASE_URL=
DATABASE_URL_BASE=
NETWORK=mainnet
S3_ENDPOINT=
S3_ACCESS_KEY=
S3_SECRET_KEY=

# Optional
DATABASE_CA=
DATABASE_CERT=
DATABASE_KEY=
CONTRACT_START_BLOCK=
S3_REGION=
S3_BUCKET=
SENTRY_DSN=
```
